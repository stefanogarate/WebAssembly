# WebAssembly Test

Contiene ejemplos de pruebas para WebAssembly


## Docker

### Build
```
$ docker-compose build
```

### Run
```
$ docker-compose up -d
```

## Develop

1. Desarrolla en tu maquina local dentro de `./src`
2. Complila en contenedor `wasm`
    ```
     $ docker exec -it wasm bash
    ```
